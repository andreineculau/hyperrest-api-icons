HyperREST API icons

@andreineculau (cont.) I, icon designer, will do it myself hyperrest.github.io/api-icons #hyperrest #api #icons

— Andrei Neculau (@andreineculau) March 18, 2013

This is a first quick go at creating a line of REST/HTTP icons based on Font Awesome, but more important than the icons are the CSS class names (the icon API, if you will).

If you want to add or change CSS class names, raise an issue.
If you create another line of icons following this naming pattern, I'll be happy to link to it. Raise an issue or make a pull request. 

/* CSS Usage */
@import url('//netdna.bootstrapcdn.com/font-awesome/4.1.0/css/font-awesome.min.css');
@import url('//rawgithub.com/hyperrest/api-icons/v0.1.0/hyperrest-api-icons.css');
